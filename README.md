# Autonomous Data Preparation Engine

## Project Goal
Build a full-stack web application that allows users to upload raw datasets, automatically analyze and clean them, detect data quality issues, engineer features, and output a fully ML-ready dataset.

## Tech Stack
- **Frontend**: React (Vite)
- **Backend**: Node.js + Express
- **ML Service**: Python (FastAPI)
- **Database**: MongoDB

## Structure
- `/frontend`: React application
- `/backend`: Node.js API
- `/ml-service`: Python FastAPI service
